All key formulas from Lecture 5
Linear regression
Prediction
Ŷ = W₀ + W₁X₁ + W₂X₂ + ... + WₙXₙ
W₀ = intercept (bias); W₁…Wₙ = feature weights
Linear regression
Gradient for W₀ (intercept)
∂J/∂W₀ = (1/N) × Σ(ŷᵢ − yᵢ)
Average of all prediction errors across N data points
Linear regression
Gradient for Wⱼ (feature weight)
∂J/∂Wⱼ = (1/N) × Σ(ŷᵢ − yᵢ) × xᵢⱼ
Error × feature value, averaged over all points
Gradient descent
Parameter update rule
W_new = W_old − α × gradient
α = learning rate. Apply to every weight each iteration.
Logistic regression
Sigmoid / logit function
σ(z) = 1 / (1 + e^(−z)) z = θ₀ + θ₁X₁ + ... + θₙXₙ
Output always in (0,1); interpreted as probability of class 1
Logistic regression
Decision rule (default threshold)
σ(z) ≥ 0.5 → predict class 1 σ(z) < 0.5 → predict class 0
Threshold = 0.5 by default (corresponds to z = 0)
Loss function
Log loss — single data point
L = −y·log(ŷ) − (1−y)·log(1−ŷ)
Binary cross-entropy. Convex — works reliably with gradient descent.
Loss function
Cost function J(θ) over all N points
J(θ) = −(1/N) Σ[yᵢ·log(ŷᵢ) + (1−yᵢ)·log(1−ŷᵢ)]
Minimize this using gradient descent to train the model
Classification metrics
Accuracy
Accuracy = (TP + TN) / (TP+TN+FP+FN)
Warning: misleading on imbalanced datasets
Classification metrics
Precision
Precision = TP / (TP + FP)
Of all I predicted positive, how many actually are?
Classification metrics
Recall (sensitivity)
Recall = TP / (TP + FN)
Of all actual positives, how many did I correctly catch?
Classification metrics
F1 score
F1 = 2 × (P × R) / (P + R)
Harmonic mean of precision & recall. Use when both matter equally.
Classification metrics
F-beta score (industry)
Fβ = (1+β²)×(P×R) / (β²×P + R)
β>1: recall matters more. β<1: precision matters more.
Odds ratio (interpretation)
Coefficient interpretation
Odds ratio = e^β
e.g. β=0.3 → odds ratio 1.35 → 35% increase in odds per unit increase in X (holding others constant)
